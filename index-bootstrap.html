Mari membuat tampilan web yang menarik dan rapi!

📘 Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa dasar untuk membuat struktur halaman web. Setiap elemen ditulis menggunakan tag seperti <h1>, <p>, dan <a>.

🎨 Fungsi CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan visual seperti warna, ukuran, dan tata letak agar halaman lebih indah dan responsif.

🧩 Cara Kerja

HTML membuat kerangka, sedangkan CSS mempercantik tampilannya. Kombinasi keduanya menghasilkan web yang menarik dan mudah dibaca.

💡 Kesimpulan

Dengan menguasai HTML dan CSS, kita dapat membuat tampilan web profesional sesuai kebutuhan proyek.
